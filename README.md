

# Salary Analytics & Interactive Dashboard (Excel + Power Query + Formulas)

This project analyzes over **300,000 job postings** to uncover insights about salary ranges, in-demand skills, job types, and recruitment platforms.  
It demonstrates a complete analytics workflow using **Power Query**, **advanced Excel formulas**, and **interactive dashboards**, supported by hidden cleaned datasets.

## Project Overview

This project reflects a real-world data analytics pipeline:

### **Raw Data Import**
Over 81k job records were imported into Excel.

### **Data Cleaning with Power Query**
- Removed duplicates  
- Standardized text columns  
- Cleaned salary formats  
- Changed data types  
- Split/merged columns  
- Loaded cleaned data into hidden backend sheets  

### **Data Enhancement with Excel Formulas**
Additional calculated fields were created using:
- Lookup formulas  
- Conditional logic  
- Aggregations  
- Text transformations  
- Date and time calculations  

These enriched datasets support the dashboard.

### **Interactive Dashboard Creation**
Built using:
- Pivot Tables  
- Pivot Charts  
- Bar charts  
- Combo bar + line chart  
- Map visual  
- Slicers for Job Titles & Job Types  

### **Insights & Storytelling**
The dashboards highlight:
- Top-paying skills  
- Most in-demand skills  
- Jobs by location  
- Salary trends by skill  
- Most common job platforms
